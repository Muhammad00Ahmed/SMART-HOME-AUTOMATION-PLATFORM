## üöÄ Getting Started

### Prerequisites

- Node.js >= 20.0.0
- Python >= 3.11
- MongoDB >= 6.0.0
- Redis >= 7.0.0
- MQTT Broker (Mosquitto)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Muhammad00Ahmed/SMART-HOME-AUTOMATION-PLATFORM.git
cd SMART-HOME-AUTOMATION-PLATFORM
```

2. **Install dependencies**

Backend:
```bash
cd backend && npm install
cd ../iot-service && pip install -r requirements.txt
```

Frontend:
```bash
cd frontend && npm install
```

3. **Environment Configuration**

Backend `.env`:
```env
NODE_ENV=development
PORT=5000
MONGODB_URI=mongodb://localhost:27017/smarthome
REDIS_URL=redis://localhost:6379
MQTT_BROKER=mqtt://localhost:1883
JWT_SECRET=your_jwt_secret

# Voice Assistants
ALEXA_CLIENT_ID=your_alexa_client_id
GOOGLE_CLIENT_ID=your_google_client_id

# Weather API
WEATHER_API_KEY=your_weather_api_key
```

4. **Start services**
```bash
docker-compose up -d
```

5. **Access the platform**
- Dashboard: http://localhost:3000
- Devices: http://localhost:3000/devices
- Automation: http://localhost:3000/automation
- API: http://localhost:5000
